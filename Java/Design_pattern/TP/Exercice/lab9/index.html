Lab 9 Proxy Pattern

<< lab9.zip >>

The proxy pattern allows one object to stand in place for another.

In UML:


We will illustrate the proxy pattern by examining remote method invocation - or RMI. A Java feature that allows  one Java  program running in one JVM to invoke a method in another (i.e. remote) JVM.



The Lab

Using the source files provided in lab9.zip, make the necessary modifications to provide a remote joke server. That is,  provide a remote method with the following API:

public String getJoke();

The getJoke() method will return a random joke. (Perhaps the best way to do this is to have 4-6 jokes stored in an array and then generate a random number representing a position in the array.)

This will require making  substantial changes to virtually all files you have been provided.

You are encouraged to work on this lab with one partner. This will allow you and your partner to serve separately as the client and the server.
