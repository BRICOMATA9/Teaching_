    Téléchargement du logiciel aspiMAGIC


Ce logiciel est facile à utiliser, et très utile pour tout le monde.


aspiMAGIC permet de télécharger des sites sur le net sans pour autant être sur le site web. Pour cela aspiMAGIC est la pour vous aider. Il suffit de rentrer l'adresse a té écharger avec ou sans nom de fichier et aspiMAGIC fait le travail pour vous. Il vous permet de récuperer des types de fichiers particulier en les choisissants dans les préférences (audio, video, text, applications,programmes) . Sinon il y a la possibilité de rester sur un même site ou vous balader dans d'autre site juste en cliquant sur un changement de site.

Pour commencer, entrer une adresse internet dans la page centrale de aspiMAGIC (dans le champ "adresse").
Si un nom de fichier doit être insérer, entrer-le dans le champ "nom fichier".

Ensuite régler les "préférences" dans le menu "option";



Vous pouvez choisir de télécharger tous les types de fichiers ou quelques uns en les selectionnant dans la fenêtre "extensions".



Vous pouvez limiter aussi le nombre de téléchargement ainsi que le changement de site en selectionnant dans la fenêtre "propriété" ces cases.


Grâce au signet de aspiMAGIC vous pouvez enregistrer vos adresses favorites.
Il vous suffit d'utiliser le signet dans le menu "option".



La gestion des fichiers télécharger est aussi facile. Un répértoire download a été créer où son stocké tous les fichies récupéré et ce répertoire est visualisable dans une fenêtre séparé nommé "hierachie".



Nous avons aussi créer un browser web pour vous aider à visualiser vos fichiers télécharger. Voici VisuMagic
